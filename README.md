# DogCare

## Overview
DogCare is a web application designed to manage daily dog-care activities such as walks, feeding, medical treatments, and reminders.  
The system allows users to add actions through forms, validate input on the client side using JavaScript and jQuery, and store the data in a MySQL database using PHP.  
The interface is responsive and built with HTML, CSS, and Bootstrap, enabling convenient viewing and management of dog-care information and activity history.

## Technologies
- HTML5
- CSS3
- Bootstrap
- JavaScript
- jQuery
- PHP
- MySQL
